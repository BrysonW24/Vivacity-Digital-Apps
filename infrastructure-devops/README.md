## Quick Start

### Docker Deployment

```bash
cd infrastructure-devops/containers/docker

# Build container
docker build -t vivacity-app .

# Run locally
docker-compose up
```

### Kubernetes Deployment

```bash
cd infrastructure-devops/kubernetes

# Apply manifests
kubectl apply -f manifests/

# Or use Helm
helm install vivacity-app ./helm/
```
