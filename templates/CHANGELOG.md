## Version Guidelines

### Version Number Format: MAJOR.MINOR.PATCH

- **MAJOR**: Breaking changes, major restructuring
- **MINOR**: New templates, significant enhancements
- **PATCH**: Bug fixes, minor updates, documentation improvements

### Types of Changes

- **Added**: New templates, features, or capabilities
- **Changed**: Changes to existing templates
- **Deprecated**: Templates scheduled for removal
- **Removed**: Deleted templates or features
- **Fixed**: Bug fixes and corrections
- **Security**: Security-related changes

## Deprecation Policy

Templates will be deprecated following this process:

1. **Announcement**: Deprecated status added to documentation
2. **Grace Period**: Minimum 6 months before removal
3. **Migration Guide**: Provided for alternative solutions
4. **Removal**: After grace period in next major version
