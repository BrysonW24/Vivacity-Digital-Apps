## Quick Start

### New Developer Onboarding

```bash
# Start with onboarding resources
cd development-resources/onboarding
cat README.md

# Setup development environment
cd setup
./setup-script.sh
```

### Learning Flutter

```bash
# Flutter learning path
cd development-resources/building-with-flutter

# Start with Dart fundamentals
cd 0-dart-language

# Move to Flutter architecture
cd ../1-flutter-architecture

# Learn cloud integration
cd ../2-cloud-integration

# Build sample apps
cd ../3-apps
```

### Explore Examples

```bash
# Browse Flutter examples
cd development-resources/examples

# Run an example
cd hello_world
flutter run
```
